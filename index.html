Unordered List

  • With an unordered list, you get the same indentation and lineup, but just no numbering.
  • This can be a list of anything that has no specific order.
  • The code uses the same 'li' list item tags as the ordered list.
  • You can change the style of the bullets for each list item as well in CSS.
